# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ShiftScope, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: **security@shiftscope.dev**

We will acknowledge receipt within 48 hours and provide a detailed response within 7 days.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Disclosure Policy

- We follow coordinated disclosure practices.
- We aim to release patches within 30 days of confirmed vulnerabilities.
- Credit will be given to reporters unless they prefer anonymity.
